# Balieternatours Landing Page

A cinematic tour & travel landing page built with Next.js, Tailwind CSS, and HeroUI (NextUI v2). The experience blends luxury travel storytelling with light/dark theming, bento grids, and a video hero showcasing Bali.

## Getting started

```bash
pnpm install
pnpm dev
```

If your environment blocks network installs, set your npm registry/proxy accordingly before installing dependencies.
